Selectors in Html

Types of Selector

  1. CSS element selector
  2. CSS Id selector
  3. CSS Class selector
  4. The CSS Group Selector

Properties of it's type

  • In Element selector all element of same type are selected and the style is given to the selected element , example p,hr,h,etc element if we select p that is paragraph all the paragraph in the program will get the same style
  • In ID selector , as we know id is unique so the style of every id will be differnt
  • In case of Class , If in some condition we need some comman propeties in same type of element or different type of element we assigend the same name of class to the elements and the assigned the style to the class.
